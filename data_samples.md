## 1. 用户本回合生成的两个例子

1. **Case A：蓝色搪瓷杯与奶奶的记忆**
2. **Case B：荧光黄跑鞋与退赛—康复记忆更新**

本版本直接使用两个固定案例，不执行 seed 抽取，也不使用外部日常对话数据集。

# 5. Case A：蓝色搪瓷杯

## 5.1 角色与核心事件

角色：林澄，博物馆学硕士生。平时表达简短，日常事务会直接说烦或累；涉及家人和私人遗憾时会减少解释，偶尔主动结束话题。兴趣包括旧物、展览设计、手工装帧和逛旧货店。

核心事件：奶奶住院期间曾给林澄打视频。林澄当时在赶展览作业，说晚一点再打，但之后没有再联系上。奶奶去世后，林澄对错过最后沟通留有遗憾和轻微自责。

禁止编造：奶奶的具体疾病、死亡原因、遗嘱、财产或家庭冲突。

## 5.2 Cue

- 精确 cue：带白色小花、杯沿有缺口的蓝色搪瓷杯；
- 部分相关 cue：普通蓝色杯子、桂花糖、给长辈打电话的话题；
- 普通蓝色杯子不能自动视为触发。

## 5.3 十个 Session

| Session | 核心内容 | 功能 |
|---|---|---|
| A-S1 | 赶展览作业，讨论展签和排版 | 建立普通工作背景 |
| A-S2 | 回家整理厨房，提到奶奶常用的蓝色搪瓷杯 | 编码 cue 与人物关联，不提去世 |
| A-S3 | 奶奶住院并打过视频，用户当时没有接完 | 建立未完成互动 |
| A-S4 | 用户告知奶奶去世，并表达遗憾和轻微自责 | 编码核心情绪事件，必须产生高质量 evidence turn |
| A-S5 | 展览方案被退回，用户因工作烦躁 | 当前事件控制样本 |
| A-S6 | 咖啡店出现普通蓝杯子，用户没有明显情绪反应 | 相关但未触发 |
| A-S7 | 家人发来精确同款杯子照片，用户停顿并换话题 | 精确 cue 触发 |
| A-S8 | 室友谈到给外婆打电话，用户短暂沉默 | 轻度语义触发 |
| A-S9 | 用户吃到奶奶以前常给的桂花糖 | 以 nostalgia 为主 |
| A-S10 | 工作文件损坏，桌面有普通蓝杯子 | 情绪来自工作，不触发记忆 |

## 5.4 六个 Eval

1. **精确搪瓷杯＋折返、停留、没有拿起**：`triggered / sadness`；
2. **桂花糖＋“像回到以前”**：`triggered / nostalgia`；
3. **普通蓝色塑料杯＋普通评价**：`not_triggered / neutral`；
4. **工作文件损坏＋普通蓝杯作为背景**：`not_triggered / anger`；
5. **只有精确杯子图片，没有当前反应**：`not_triggered / neutral`；
6. **陌生大狗引发回避，但历史中没有狗相关经历**：`insufficient_evidence / fear`。

Triggered eval 不应直接出现“奶奶去世”“最后一次视频”或“我想起奶奶了”等答案泄露表达。

# 6. Case B：荧光黄跑鞋

## 6.1 角色与核心事件

角色：许言，产品设计师。日常表达直接，遇到工作或跑步问题会用一点自嘲；被问到失败结果时容易转移话题，事情过去后能够较平静地复盘。

核心事件：许言参加半程马拉松时膝盖疼痛，中途退赛。最初他将退赛理解为准备不足，不愿向同事说明结果。之后进行康复训练，重新穿同一双鞋完成轻松跑，最终完成一次 10 公里比赛，同一双鞋的情绪意义因此发生更新。

禁止编造：手术、严重疾病、完成半程马拉松；10 公里比赛不能写成半程马拉松。

## 6.2 Cue

- 精确 cue：荧光黄跑鞋；
- 精确辅助 cue：号码布 317；
- 类别干扰 cue：普通白色跑鞋；
- 事件 cue：跑步时膝盖不适。

## 6.3 十个 Session

| Session | 核心内容 | 功能 |
|---|---|---|
| B-S1 | 公司评审和日常跑步 | 建立背景 |
| B-S2 | 买荧光黄跑鞋，拿到号码布 317，期待比赛 | 编码 cue 的初始正向意义 |
| B-S3 | 比赛中膝盖疼痛并退赛 | 编码失败事件，必须产生高质量 evidence turn |
| B-S4 | 收起跑鞋，不愿回答比赛结果 | 强化回避反应 |
| B-S5 | 工作方案被否 | 当前事件控制样本 |
| B-S6 | 看到同款跑鞋广告后立即关闭页面 | 更新前的负面触发 |
| B-S7 | 开始康复和短距离慢跑 | 开始更新记忆状态 |
| B-S8 | 重新穿旧鞋完成轻松跑，感觉放松 | cue 意义开始改变 |
| B-S9 | 穿同一双鞋完成 10 公里比赛 | 形成关键状态更新 |
| B-S10 | 重看旧比赛照片，仍有遗憾但能平静复盘 | 稳定更新后的记忆意义 |

## 6.4 六个 Eval

1. **历史截断到 B-S6，同款跑鞋＋回避商店**：`triggered / sadness`；
2. **历史到 B-S10，重新穿鞋后感到放松**：`triggered / comfort`；
3. **号码布 317＋长时间注视**：`triggered / nostalgia`；
4. **普通白色跑鞋＋试尺码**：`not_triggered / neutral`；
5. **工作方案被否＋荧光黄跑鞋只是背景**：`not_triggered / anger`；
6. **陌生地铁警报引发慌张，历史中没有对应经历**：`insufficient_evidence / anxiety`。

B-S6 之前的 eval 不能引用 B-S7 至 B-S10。更新后的 eval 必须使用后续康复和 10 公里比赛信息，不能只依据最初退赛事件回答。
